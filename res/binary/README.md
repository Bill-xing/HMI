# LeArm STL Assets

This directory is reserved for local Hiwonder LeArm STL model files used by the Qt/OpenGL digital twin view.

The STL files are not redistributed in this repository. To run the 3D visualization locally, place authorized LeArm STL files here with these names:

```text
base_link.STL
link_1.STL
link_2.STL
link_3.STL
link_4.STL
link_5.STL
```
